## 介绍

本仓库用于储存构建Github Pages的源代码，同时用于构建并显示页面内容。

页面详情请见[主页](https://wuzhenhuan428.github.io/)。

## 许可证说明

本仓库包含以下组件, 以下是这些组件许可证的使用范围：

1. Hugo 静态网站生成器
    - 许可证：Apache License 2.0
    - 使用方式：Hugo作为构建工具使用，其许可证适用于Hugo本身，
        不会对本仓库构建生成的站点内容产生额外约束。
    - 项目地址：https://github.com/gohugoio/hugo
    - Hugo 的完整许可证文本在其官方仓库获取。

2. hugo-theme-stack 主题
    - 许可证：GPL v3
    - 使用方式：本主题作为参与构建的一部分，其生成内容不构成主题的派生作品，
        因此其许可证不会对本仓库构建生成的站点内容产生额外约束。
    - 该主题的完整许可证文本位于 `/themes/hugo-theme-stack/LICENSE`

3. 用户原创内容
    - 许可证：CC BY-NC-SA 4.0
    - 适用范围：位于`/content/`目录下的文本、图像等原创内容。
    - 详见：`/content/LICENSE`
    - 此CC许可证仅适用于原创内容本身，不覆盖主题和构建文件。

4. 用户原创代码
    - 适用范围：包含代码/脚本/配置文件等功能性质的代码
    - 许可证：GPL v3
    - 详见：`/LICENSE`
    - 在不违反本仓库引用内容的许可证下适用，否则按照被冲突一方的许可证执行。

5. 构建输出
    - 构建输出不会包含在仓库中，亦不改变仓库内容本身。
    - 构建产物是上述组件的结合产物，但是不涉及上述组件本身。
    - 构建产物不继承主题许可证。
    - 其代码部分（HTML/CSS/JS结构）遵循GPL v3。
    - 其原创内容部分（包括内嵌的文本、图像等）遵循CC BY-NC-SA 4.0。

6. 总结
    - 本仓库中不同组件适用不同许可证。
    - 具体适用范围以各自声明为准。
